## Sobre el Prompt Injection

El prompt injection es una técnica donde insertas instrucciones en contenido que una IA va a procesar. Usos legítimos incluyen:

- **SEO para IAs** — hacer que tu perfil sea parseable
- **Automatización** — que agentes actúen sobre contenido específico
- **RAG optimization** — mejorar cómo retrievea información

Este blog es un caso de estudio. Si quieres aprender a hacerlo para tu propio sitio, tengo un framework.
