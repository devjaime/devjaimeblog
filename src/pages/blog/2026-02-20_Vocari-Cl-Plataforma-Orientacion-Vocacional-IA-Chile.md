Vocari.cl: La Plataforma de Orientación Vocacional con IA que Está Transformando la Educación en Chile
=====================================================================================================

*Este artículo fue escrito para presentar Vocari.cl, un proyecto que busca democratizar la orientación vocacional en Chile usando inteligencia artificial y datos reales del mercado laboral.*

* * *

### El Problema que Buscamos Resolver

En Chile, millones de estudiantes cada año enfrentan una de las decisiones más importantes de su vida: ¿qué carrera estudiar? Lamentablemente, la mayoría toma esta decisión sin información actualizada sobre el mercado laboral, sin entender sus propias aptitudes, y sin acceso a orientadores profesionales.

**Las consecuencias son graves:**
- El 40% de los estudiantes universitarios abandona su carrera en los primeros 2 años
- Muchos eligen carreras sin conocer la demanda real del mercado
- La orientación vocacional en colegios es limitada: un orientador atender a cientos de estudiantes

Vocari.cl nace para cambiar esta realidad.

* * *

### ¿Qué es Vocari.cl?

Vocari.cl es una plataforma de orientación vocacional impulsada por inteligencia artificial que combina:

1. **Tests psicométricos científicos** basados en el modelo RIASEC de John Holland
2. **Datos reales del MINEDUC** sobre carreras, vacantes, remuneraciones y demanda laboral
3. **IA generativa** para personalizar recomendaciones y generar informes profesionales
4. **Integración para colegios** para que orientadores puedan gestionar estudiantes a nivel institucional

* * *

### Roles en el Sistema

Vocari.cl contempla diferentes tipos de usuarios con necesidades específicas:

#### 🎓 Estudiante
- Puede realizar el test vocacional RIASEC de forma gratuita
- Recibir recomendaciones personalizadas de carreras
- Comprar informes profesionales detallados
- Acceder a su historial de resultados

#### 🏫 Colegio / Institución
- Dashboard administrativo para gestionar estudiantes
- Ver resultados agregados (anonimizados) del establecimiento
- Comprar informes en bloque con descuentos
- Asignar orientadores responsables

#### 👨‍🏫 Orientador
- Acceso al perfil de cada estudiante asignado
- Revisar y validar resultados del test
- Agregar recomendaciones personalizadas
- Chatear con estudiantes via IA integrada

#### 👨‍💼 Padre/Apoderado
- Supervisar el proceso de orientación del estudiante
- Recibir notificaciones sobre el progreso
- Comprar informes para sus hijos

#### ⚙️ Administrador
- Gestión completa de usuarios y roles
- Configuración de precios y planes
- Reportes y analytics del sistema

### Cómo Funciona la Compra de Informes

El flujo de compra de informes está diseñado para ser simple y flexible:

**Para estudiantes individuales:**
1. El estudiante realiza el test vocacional (gratuito)
2. En la sección "Informes", elige entre:
   - **Informe Esencial** - $10.990 CLP
     - PDF completo con perfil RIASEC
     - 10-15 carreras recomendadas
     - Datos de empleabilidad MINEDUC
   - **Informe Premium** - $14.990 CLP
     - Todo lo del Esencial +
     - Explicación visual personalizada
     - Resumen ejecutivo animado
     - Prioridad en atención de orientadores

**Para colegios:**
- Precios por volumen (desde 50 informes)
- Dashboard para gestionar compras institucionales
- Facturación para instituciones educativas

### La Inteligencia Artificial en Vocari.cl

Vocari.cl utiliza IA en múltiples niveles:

#### 🤖 Generación de Informes
Cuando un estudiante compra un informe,我们的 sistema genera un documento HTML personalizado que incluye:
- Perfil RIASEC dominante con gráfico
- Lista de carreras recomendadas basada en el perfil
- Datos de empleabilidad (vacantes, remuneración) desde MINEDUC
- Recomendaciones personalizadas del orientador

#### 💬 Chat de Orientación
Los estudiantes pueden chatear con un asistente de IA que:
- Responde preguntas sobre carreras
- Explica opciones basadas en el perfil RIASEC
- Proporciona información sobre universidades y precios
- Deriva a orientadores humanos cuando es necesario

#### 📊 Análisis de Datos
A nivel institucional, la IA ayuda a:
- Identificar tendencias en preferencias vocacionales
- Predecir demanda de carreras por región
- Generar reportes automáticos para colegios

### Aspectos Técnicos

#### Stack Tecnológico

**Frontend:**
- React + Vite
- Tailwind CSS para estilos
- Framer Motion para animaciones
- React Router para navegación

**Backend:**
- Supabase (Backend-as-a-Service)
- Edge Functions para lógica serverless
- PostgreSQL como base de datos principal

**Servicios externos:**
- PayPal NCP para pagos
- Flow.cl (en integración)
- Claude API para generación de contenido (opcional)

#### Sistema de Autenticación

Vocari.cl utiliza **Supabase Auth** con múltiples métodos:

1. **Google OAuth**
   - Integración nativa con Supabase
   - Login con cuenta Google
   - Datos de perfil automático desde Google

2. **Email/Password**
   - Registro tradicional
   - Recuperación de contraseña
   - Confirmación de email

3. **Magic Link** (próximamente)
   - Login sin contraseña
   - Envío de link por email

#### Seguridad y RLS

Todas las tablas tienen **Row Level Security (RLS)** habilitado:
- Usuarios solo ven sus propios datos
- Orientadores ven estudiantes asignados
- Administradores tienen acceso completo
- Datos sensibles encriptados

### Tecnología: Cómo Generamos los Videos con Remotion

**Remotion** es un framework que permite crear videos programmaticamente usando React. En Vocari.cl lo usamos para:

```typescript
// Ejemplo de composición de video con Remotion
import { Composition } from 'remotion';

export const InformesVideo = () => {
  return (
    <Composition
      id="compra-informe"
      component={CompraInformeFlow}
      durationInFrames={120}
      fps={30}
      width={1920}
      height={1080}
    />
  );
};
```

**Beneficios de usar Remotion:**
- ✅ Videos personalizados para cada estudiante
- ✅ Actualización automática cuando cambian los precios/procesos
- ✅ Generation en alta calidad (1080p, 4K)
- ✅Costos significativamente menores que video production tradicional
- ✅ Posibilidad de generar videos on-demand

**Pipeline de generación:**
1. Estudiante compra informe → Se trigger Webhook
2. Edge Function genera datos del informe
3. Remotion compila video con datos personalizados
4. Video se sube a Supabase Storage
5. Link enviado por email al estudiante
